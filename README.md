# Johannes Maboudou

Ich entwickle praxisnahe Bewerberprojekte für duale Studiengänge in Wirtschaftsinformatik, Informatik, Data Science und KI. Dabei verbinde ich technische Grundlagen mit wirtschaftlichen Fragestellungen: Daten verarbeiten, Prozesse nachvollziehbar modellieren, einfache Software sauber strukturieren, Tests schreiben und Ergebnisse verständlich dokumentieren. Alle Projekte nutzen synthetische Daten und sind als Lern- und Bewerbungsprojekte aufgebaut. Ziel ist ein strukturiertes GitHub-Portfolio, das meine Interessen an Softwareentwicklung, Datenanalyse, Mobility, Energie, Prozessdaten und Finanz-IT sichtbar macht.

## Fokusbereiche

- Wirtschaftsinformatik und Business-/IT-Schnittstellen
- Informatik, Java/OOP und Backend-Grundlagen
- Data Science, KI-Grundlagen und technische Sensordaten
- einfache Prozess- und Datenlogik
- Energie, Nachhaltigkeit und Billing
- Mobility, Logistik, Dispatching und Prozessanalyse
- Finanz- und Versicherungs-IT
- Testing, GitHub Actions und verständliche Dokumentation
