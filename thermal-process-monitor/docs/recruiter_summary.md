# Recruiter Summary - Thermal Process Monitor

## Kurzbeschreibung

`thermal-process-monitor` ist ein technisches Bewerberprojekt zur Simulation industrieller Prozessüberwachung. Es verarbeitet synthetische Sensordaten, erkennt Grenzwertverletzungen und erstellt einen kompakten Prozessbericht.

## Warum das Projekt relevant ist

Das Projekt zeigt grundlegende Fähigkeiten in Softwareentwicklung, Datenverarbeitung, technischer Logik und Dokumentation. Es ist besonders passend für technische Informatik, Mechatronik, Maschinenbau-nahe Softwareentwicklung und industrielle IT.

## Sichtbare Kompetenzen

- Python-Projektstruktur
- CSV-Datenverarbeitung
- Validierung von Messdaten
- technische Kennzahlen
- Grenzwert- und Alarmlogik
- automatisierte Tests mit pytest
- GitHub Actions CI
- C++ Grundlagen
- verständliche Dokumentation

## Gesprächspunkte

- Wie Sensordaten geladen und geprüft werden
- Warum Grenzwerte für Temperatur, Druck und Durchfluss sinnvoll sind
- Wie aus Rohdaten ein Bericht entsteht
- Warum Tests für technische Logik wichtig sind
- Wie Python und C++ zusammen in technischen Projekten vorkommen können

## Einordnung

Level: 2

Begründung: Das Projekt ist sauber strukturiert, getestet, dokumentiert und besitzt einen klaren technischen Bewerbungsbezug. Für Level 3 könnten später Visualisierungen, konfigurierbare Grenzwerte und ein Daten-Generator ergänzt werden.
